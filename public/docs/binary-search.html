DSA Binary Search

Binary Search is an efficient search algorithm that finds the position of a target value within a sorted array. It works by repeatedly dividing in half the portion of the list that could contain the item, until you've narrowed down the possible locations to just one.

How Binary Search Works

Binary search works on the principle of Divide and Conquer. For this algorithm to work, the data collection should be in a sorted form.

  1. Find the middle element of the array.
  2. If middle element is equal to the target value, return index.
  3. If middle element is greater than target, search in the left half.
  4. If middle element is less than target, search in the right half.
  5. Repeat until the element is found or the search space is empty.

Implementation

// Binary Search Implementation in JavaScript
function binarySearch(arr, x) {
    let low = 0;
    let high = arr.length - 1;
    let mid;

    while (low <= high) {
        mid = Math.floor((low + high) / 2);

        if (arr[mid] === x) {
            return mid;
        } else if (arr[mid] < x) {
            low = mid + 1;
        } else {
            high = mid - 1;
        }
    }
    return -1;
}

const data = [10, 20, 30, 40, 50, 70, 80, 90];
const target = 70;
console.log("Element found at index:", binarySearch(data, target));

Complexity Analysis

  • Time Complexity: O(log n). With each comparison, the search space is halved.
  • Auxiliary Space: O(1) for iterative implementation, O(log n) for recursive implementation (due to stack space).
