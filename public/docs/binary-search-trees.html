Binary Search Trees (BST)

A Binary Search Tree is a node-based binary tree data structure which has the following properties: The left subtree contains only nodes with keys lesser than the node's key. The right subtree contains only nodes with keys greater than the node's key.

Why use a BST?

A BST provides O(log n) time complexity for Search, Insert, and Delete operations in the average case. It combines the fast insertion/deletion of a linked list with the fast search capabilities of a sorted array (using binary search logic).

Searching in a BST

To search for a value:

  1. Start at the root.
  2. If the target value matches the root, return it.
  3. If target is less than the root, search the left subtree.
  4. If target is greater than the root, search the right subtree.
