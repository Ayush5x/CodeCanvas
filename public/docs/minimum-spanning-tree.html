Minimum Spanning Tree (MST)

A Spanning Tree of a connected, undirected graph is a subgraph that is a tree and includes all the vertices of the graph. A graph can have many spanning trees.

A Minimum Spanning Tree (MST) is the spanning tree with the minimum possible total edge weight.

Applications of MST

  • Network Design: Laying out telecommunication networks, electrical grids, or water pipes where you want to connect all nodes with the absolute minimum amount of wire/pipe.
  • Approximation Algorithms: Used in algorithms to solve hard problems like the Traveling Salesperson Problem (TSP).
  • Cluster Analysis: Grouping similar points in machine learning algorithms.

Key Properties

  1. An MST has exactly `V - 1` edges (where V is the number of vertices).
  2. It has absolutely no cycles (it's a tree!).
  3. If all edge weights in a graph are unique, there is only one unique MST for that graph.
