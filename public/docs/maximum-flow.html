Maximum Flow

The Maximum Flow problem asks: Given a flow network with a single source node and a single sink (destination) node, what is the maximum amount of "stuff" (water, data, cars) that can flow from the source to the sink per unit of time?

Flow Networks

A flow network is a directed graph where each edge has a capacity. You cannot push more flow through an edge than its capacity allows.

  • Source Node (s): The node where flow originates. It only has outgoing edges.
  • Sink Node (t): The destination node where flow is collected. It only has incoming edges.
  • Conservation of Flow: For every node other than the source and sink, the amount of flow entering the node must exactly equal the amount of flow exiting the node. (Water can't miraculously disappear or appear at intersections).

Applications

  • Optimizing traffic routing in road networks.
  • Maximizing data bandwidth in telecommunications.
  • Bipartite matching problems (e.g. matching job applicants to open positions).
  • Scheduling airline flight crews.
