DSA Counting Sort

Counting Sort is a non-comparison based sorting algorithm that sorts the elements of an array by counting the number of occurrences of each unique element in the array. It is efficient when the range of input data is not significantly larger than the number of elements to be sorted.

How Counting Sort Works

  1. Find the maximum element in the given array.
  2. Initialize a count array of size max + 1 with all zeros.
  3. Store the count of each element at its corresponding index in the count array.
  4. Modify the count array such that each element at each index stores the sum of previous counts. This gives the position of each element in the output array.
  5. Output the each element from the input array into the output array based on count array positions.

Implementation

// Counting Sort Implementation in JavaScript
function countingSort(arr) {
    let n = arr.length;
    let max = Math.max(...arr);
    let count = new Array(max + 1).fill(0);
    let output = new Array(n);

    for (let i = 0; i < n; i++) {
        count[arr[i]]++;
    }

    for (let i = 1; i <= max; i++) {
        count[i] += count[i - 1];
    }

    for (let i = n - 1; i >= 0; i--) {
        output[count[arr[i]] - 1] = arr[i];
        count[arr[i]]--;
    }

    return output;
}

const data = [4, 2, 2, 8, 3, 3, 1];
console.log("Sorted Array:", countingSort(data));

Complexity Analysis

  • Time Complexity: O(n + k), where n is the number of elements and k is the range of the input.
  • Auxiliary Space: O(n + k).
  • Suitability: Best used when the range of input values is small.
