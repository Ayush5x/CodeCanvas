DSA Trees

A Tree is a hierarchical data structure consisting of nodes connected by edges, typically starting from a single root node and expanding downwards.

Tree Structure

Unlike Arrays or Linked Lists, which are linear, a Tree is non-linear. This makes trees excellent for representing hierarchical relationships, like file systems or organizational charts.

  • Root: The topmost node of the tree.
  • Parent & Child: A node that connects to another node downward is the parent, and the node it connects to is its child.
  • Leaf: A node with no children.

Figure: A Binary Tree structure.

Binary Search Tree (BST)

A Binary Search Tree is a special type of Binary Tree where:

  1. The left child's value is always strictly less than its parent's value.
  2. The right child's value is always strictly greater than its parent's value.

This property allows for incredibly fast searching, insertion, and deletion operations (O(log n) time complexity on average).

Code Example

Implementation of a BST Insertion in JavaScript:

class TreeNode {
  constructor(value) {
    this.value = value;
    this.left = null;
    this.right = null;
  }
}

class BinarySearchTree {
  constructor() {
    this.root = null;
  }
  
  insert(value) {
    let newNode = new TreeNode(value);
    
    // If tree is empty, new node becomes root
    if (this.root === null) {
      this.root = newNode;
      return this;
    }
    
    let current = this.root;
    while (true) {
      if (value === current.value) return undefined; // No duplicates allowed
      
      // Go Left
      if (value < current.value) {
        if (current.left === null) {
          current.left = newNode;
          return this;
        }
        current = current.left;
      } 
      // Go Right
      else {
        if (current.right === null) {
          current.right = newNode;
          return this;
        }
        current = current.right;
      }
    }
  }
}
