Multi-dimensional Arrays

A multi-dimensional array is an array of arrays. The most common type is a 2D array, which can be visualized as a grid or table with rows and columns.

2D Arrays (Matrices)

A 2D array is indexed by two numbers: the row index and the column index.

// 2D Array in JavaScript
let matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
];

// Accessing the number 6 (Row 1, Column 2)
console.log(matrix[1][2]);

Visualizing a 2D Array

A 3x3 Grid (Matrix) representation.

Common Uses

  • Matrices: Mathematical calculations and transformations.
  • Game Boards: Chess, Tic-Tac-Toe, or Sudoku grids.
  • Image Processing: Images are essentially 2D arrays of pixels.
  • Geographic Data: Grids representing maps or terrain.
