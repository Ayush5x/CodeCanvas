Array Operations

Basic operations on arrays include Traversing, Inserting, Deleting, and Searching. Understanding how these operations affect performance is key to choosing the right data structure.

1. Traversing an Array

Traversal means visiting every element in the array exactly once. This is used for printing elements, calculating sums, or searching.

// JavaScript Traversal
let fruits = ["Apple", "Banana", "Cherry"];
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

Time Complexity: O(n)

2. Inserting Elements

Inserting an element at the end is fast (O(1)). However, inserting at the beginning or middle requires shifting all subsequent elements to the right to make space.

Inserting 15 between 10 and 20 requires shifting 20 to the right.

3. Deleting Elements

Similar to insertion, deleting an element from the middle requires shifting all elements to the left to fill the gap.

Time Complexity: O(n) in worst case (deleting the first element).
