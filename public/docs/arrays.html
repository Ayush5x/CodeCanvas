DSA Arrays

An array is a data structure used to store multiple elements of the same data type in a contiguous block of memory.

What is an Array?

An array is a linear data structure that collects elements and stores them in contiguous and adjacent memory locations. Arrays work on an index system starting from 0 to (n-1), where n is the size of the array.

This contiguous memory layout is what makes arrays so fast at reading data. If you know the memory address of the first element (the base address), you can instantly calculate the exact memory address of any other element just by knowing its index.

Figure: An array of 5 integers. Indices are marked in green above the boxes.

Array Operations

  • Traversal: Print all the array elements one by one.
  • Insertion: Adds an element at the given index. (Can be slow if inserting at the beginning due to shifting).
  • Deletion: Deletes an element at the given index. (Can also require shifting).
  • Search: Searches an element using the given index or by the value.

Code Example

Here is an example of creating and traversing an array in JavaScript:

// 1. Creating an array
let myNumbers = [7, 11, 3, 2, 9];

// 2. Accessing elements by index
console.log(myNumbers[0]); // Outputs: 7
console.log(myNumbers[3]); // Outputs: 2

// 3. Traversing the array
for (let i = 0; i < myNumbers.length; i++) {
  console.log("Element at index " + i + " is " + myNumbers[i]);
}

// 4. Modifying an element
myNumbers[2] = 42; // The array is now [7, 11, 42, 2, 9]
