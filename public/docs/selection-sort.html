DSA Selection Sort

Selection Sort is a simple comparison-based sorting algorithm. It works by dividing the input list into two parts: a sorted sublist of items which is built up from left to right at the front (left) of the list and a sublist of the remaining unsorted items that occupy the rest of the list.

How Selection Sort Works

The algorithm repeatedly selects the smallest (or largest, depending on sorting order) element from the unsorted sublist and moves it to the sorted sublist.

Example: [29, 10, 14, 37, 13]

  1. Find the minimum element in [29, 10, 14, 37, 13]. Minimum is 10.
  2. Swap 10 with the first element (29). Array becomes [10, 29, 14, 37, 13].
  3. Find the minimum element in the remaining unsorted part [29, 14, 37, 13]. Minimum is 13.
  4. Swap 13 with 29. Array becomes [10, 13, 14, 37, 29].
  5. Continue this process until the array is sorted.

Implementation

// Selection Sort Implementation in JavaScript
function selectionSort(arr) {
    let n = arr.length;
    for (let i = 0; i < n - 1; i++) {
        let min_idx = i;
        for (let j = i + 1; j < n; j++) {
            if (arr[j] < arr[min_idx]) {
                min_idx = j;
            }
        }
        // Swap the found minimum element with the first element
        let temp = arr[min_idx];
        arr[min_idx] = arr[i];
        arr[i] = temp;
    }
    return arr;
}

const data = [29, 10, 14, 37, 13];
console.log("Sorted Array:", selectionSort(data));

Complexity Analysis

  • Time Complexity: O(n²) in all cases (worst, average, and best) because it always finds the minimum element for each position.
  • Auxiliary Space: O(1) as it sorts the array in-place.
