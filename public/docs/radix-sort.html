DSA Radix Sort

Radix Sort is a non-comparison based sorting algorithm that sorts data with integer keys by grouping keys by the individual digits which share the same significant position and value. It processes digits from least significant to most significant or vice versa.

How Radix Sort Works

Radix sort uses counting sort as a subroutine to sort the digits.

  1. Find the maximum element in the array to know the number of digits.
  2. Do counting sort for every digit. The sorting starts from the least significant digit (unit place) and moves towards the most significant digit.
  3. After each pass of counting sort on a digit position, the array is partially sorted.

Implementation

// Radix Sort Implementation in JavaScript
function getDigit(num, i) {
    return Math.floor(Math.abs(num) / Math.pow(10, i)) % 10;
}

function digitCount(num) {
    if (num === 0) return 1;
    return Math.floor(Math.log10(Math.abs(num))) + 1;
}

function mostDigits(nums) {
    let maxDigits = 0;
    for (let i = 0; i < nums.length; i++) {
        maxDigits = Math.max(maxDigits, digitCount(nums[i]));
    }
    return maxDigits;
}

function radixSort(nums) {
    let maxDigitCount = mostDigits(nums);
    for (let k = 0; k < maxDigitCount; k++) {
        let digitBuckets = Array.from({ length: 10 }, () => []);
        for (let i = 0; i < nums.length; i++) {
            let digit = getDigit(nums[i], k);
            digitBuckets[digit].push(nums[i]);
        }
        nums = [].concat(...digitBuckets);
    }
    return nums;
}

const data = [170, 45, 75, 90, 802, 24, 2, 66];
console.log("Sorted Array:", radixSort(data));

Complexity Analysis

  • Time Complexity: O(d * (n + k)), where n is the number of elements, d is the number of digits in the maximum element, and k is the radix (usually 10).
  • Auxiliary Space: O(n + k).
  • Stability: It is a stable sorting algorithm.
